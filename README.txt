High And Low Gameの説明:


このゲームは数字を推測するゲームです。ゲームの作成にはJavascript、HTMLとCSSで作られます。答えは正解したらスコアを消さないようにスコアの値はブラウザのセッションメモリーに保存され、間違ったらスコアをゼロにリセットにされます。


条件： 
1. 画面の数字より大きいと思われたらHighボタンを押してください。
2. 画面の数字と等しいと思われたらDrawボタンを押してください。
3. 画面の数字より小さいと思われたらLowボタンを押してください。


ゲームの仕組み :


1. 数字の推測の範囲は 0 から 2までです。
2. 数字は画面に出ている数字とユーザーの数字という二つ数字をもらいますが、画面からユーザーの数字は隠しています。
3. ユーザーの数字は ゲームの画面に出ている数字とどちらが大きいのか小さいのかと推測してください。
4. 正しかったらスコアを追加されます。間違ったらスコアをゼロにリセットして最初からゲームし直します。
